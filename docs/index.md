# Universal Commerce Protocol SDK

Welcome to the documentation for the Universal Commerce Protocol (UCP) Python SDK!

This SDK gives you the typed Pydantic models needed to implement or integrate with UCP compliant endpoints in Python.

## Why UCP?

UCP is an open standard that decouples commerce infrastructure from front-end conversational interfaces. This SDK provides complete schema serialization and deserialization guarantees exactly matching your desired specification version.
